# Mohammad Mohid - Portfolio Website

Live at: [mohid.tech](https://mohid.tech)

## Deployment Guide

This portfolio is hosted on GitHub Pages with a custom domain.

### Local Development
Simply open `index.html` in your browser to view locally.

### Deployment Steps
1. Push to GitHub repository
2. Enable GitHub Pages in repository settings
3. Configure custom domain `mohid.tech`

## Tech Stack
- HTML5
- CSS3 (Custom animations)
- Vanilla JavaScript
- Hosted on GitHub Pages
